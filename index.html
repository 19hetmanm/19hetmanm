In Unit 2, I learned about HTML. HTML stands for HyperText Markup Language. HTML is used to create documents on the web and includes a variety of tags that define the structure of the document. Unit 3

In Unit 3, I learned about CSS; this stands for Cascading Style Sheets. CSS is used in conjunction with HTML to control how web pages are displayed. With CSS there are two types of Value Property; Absolute Units and Relative Units. Absolute Units include; point(pt) and pixel(px). Relative Units include; percent(%) and em(em). Between Relative and Absolute, you would use Relative because they are scalable and hence preferable to use as compared to Absolute.

Unit 5

In Unit 5, I learned about the different between Client-side and Server-side scripting. Client-side scripts are executed within the user's web browser where Server-side scripts are executed on the web server before the web page is sent to the browser. Client-side scripts can be read by the user and can be found in the source code of web pages that use them. Server-side scripts cannot be read by the user and can be found on the web server, and when requested by the web browser, they execute. The most common client-side scripting language is JavaScript, next is VBScript, most web developers use JavaScript because they are sure that their website will work in all browsers. The term progressive enhancement is used to describe the practice of creating web pages, first will CSS and then with JavaScript. In CSS you enhance the presentation and in JavaScript you enhance the functionality of the page. A database is any organized collection of information, it is used to store data. Next, I learned some JavaScript, like how to show an alert and create a clock.

Some Important Client and Server Scripts:

  1. PHP(open-source script)-The most widely used scripting language on the web
  2. Perl(open-source script)- Been around since 1987. Used extensively on websites that have been around a while.
  3. ASP.NET(server-side scripting technology)-Developed by Microsoft and only runs on Microsoft servers. ASP stands for Active Server Pages and was used widely on the web. ASP.NET and ASP are different technologies.
  4. Java(object oriented)- Used to create software applications that run over the web.Released in 1995 and quickly because the hottest new programming language. Java and JavaScript are not related.
  5. Python(open-source programming)- Designed to be easy to learn, simple, and easy to read. This would reduce the cost of program maintenance. Python is used by Google extensively.
