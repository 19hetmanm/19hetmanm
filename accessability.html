Web Accessibility Checklist
User Characteristic Accessible Design Tip
Unable to See -Code all images with ALT text
Unable to Perceive Colors -Avoid using color alone to convey information
Unable to Use Mouse -Be sure all website features can be accessed using keyboard
Unable to Hear -Add captions to multimedia
Prone to having Seizures -Avoid content that flashes
Easily Destructible -Keep the design simple

Accessibility Review of the Bordentown Regional High School Home Page

The Bordentown Regional High School website came up with 5 HTML errors. The first error was, "The charset attribute on the script element is obsolete". An error that occured twice was, "The type attribute is unnecessary for JavaScript resources". The fourth error was, "script element between head and body". The last error was, "Cannot recover after last error. Any further errors will be ignored". I don't really know how to fix the first problem but to fix the problem that occured twice, I would remove the "type" attributes from the code. Even though the website have some HTML errors, it still runs fine so the errors are too big of a struggle for the website.
